======
Python
======

Prerequisites:
--------------

- automake >=1.11.2
- Typical build tools like configure, make, gcc ...


To install the latest stable release from
https://pypi.python.org/pypi/wishbone use *pip*:

.. code-block:: sh

    $ pip install wishbone --process-dependency-links


or use *easy_install*:


.. code-block:: sh

    $ easy_install wishbone
